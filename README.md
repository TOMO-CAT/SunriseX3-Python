# SunriseX3-Python

## 简介

旭日 X3 派上的 Python 项目。

## 依赖

```bash
sudo pip3 install toml
```

## Examples

以模块的方式执行 example：

```bash
# 抓拍图片
sudo python3 -m examples.picture_capture

# 录制视频
sudo python3 -m examples.video_capture
```
